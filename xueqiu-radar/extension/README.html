安装雪球大V雷达扩展

用途：读取你在浏览器中已经正常登录并打开的雪球页面，将页面里真实的大V、粉丝数和动态同步到个人网页。

安装

1. 下载并解压扩展包。

2. Edge 打开 edge://extensions；Chrome 打开 chrome://extensions。

3. 开启“开发人员模式”。

4. 点击“加载解压缩的扩展”，选择解压后的 extension 文件夹。

使用

1. 在同一个浏览器中正常登录雪球。

2. 打开雪球推荐页、热门页、个股讨论页或大V主页。

3. 点击浏览器工具栏中的“雪球大V雷达”。

4. 点击“扫描当前雪球页面”。扩展只会把粉丝数达到1万且页面能识别到粉丝数据的用户同步到网页。

5. 返回大V雷达网页，刷新即可看到真实人物和真实动态。

边界

扩展不保存雪球密码，不绕过验证码，不在你退出登录后继续访问。雪球页面结构变化后，识别规则可能需要更新。
